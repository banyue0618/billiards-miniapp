# 班级点名系统

这是一个苹果风格的班级点名系统，专为课堂点名环节设计，旨在提高课堂互动性和趣味性。

## 特点

- 苹果风格UI，大屏幕显示
- 随机公平的点名机制
- 动画效果：名字从上往下掉落
- 网络热梗结合，增加课堂乐趣
- 音效提示，增强交互体验

## 使用方法

1. 准备学生名单文件 `students.txt`：
   - 第一行为班级名称
   - 从第二行开始，每行填写一个学生姓名

2. 准备音效文件：
   - 在 `sounds` 目录下放置两个音效文件：
     - `roll.mp3`：点名过程的音效
     - `finish.mp3`：点名结束的音效

3. 使用系统：
   - 直接打开 `index.html` 文件
   - 点击"选择学生名单文件"按钮，选择您准备好的txt文件
   - 加载成功后，点击"开始点名"按钮开始随机点名

## 学生名单文件格式示例

```
初二三班
张小明
李华
王芳
赵刚
陈思思
...其他学生
```

## 音效获取建议

可以从以下网站获取免费的音效：
- [Freesound](https://freesound.org/)
- [ZapSplat](https://www.zapsplat.com/)
- [Mixkit](https://mixkit.co/free-sound-effects/)

## 自定义

1. 修改 `script.js` 中的 `memes` 数组可以自定义网络热梗
2. 修改 `style.css` 可以调整界面样式和动画效果

## 注意事项

- 本系统使用 HTML5 的 Web API，请使用现代浏览器打开
- 全屏模式下效果最佳
- 首次使用时，浏览器可能会阻止音频自动播放，需要用户交互后才能播放音频
